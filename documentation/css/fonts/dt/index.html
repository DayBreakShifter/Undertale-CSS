Determination

Determination is the most common font in Undertale. It comes in two varieties.

  • "Determination Mono" is the font used for dialogue, and is more spaced out than its counterpart. It looks like this.
  • "Determination Sans" is the font used in menus (such as player inventory), and is slightly more tightly packed than its counterpart. It looks like this.

To use these fonts in "font-family" declarations, use "Determination Mono" and "Determination Sans", respectively.

Reccomended fallback fonts are as follows:

  • 8-bit Operator
  • FixedSys
  • monospace (for Determination Mono)
  • sans-serif (for Determination Sans)

The following classes make use of this font as their default:

  • ut-dialogue (Mono)
  • ut-menu (Sans)
