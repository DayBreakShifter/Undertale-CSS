dOcumentation - font

Hachicro

Hachicro is most well-known for being the font that damage is displayed in when you hit a monster.

It looks like this. ALL-CAPITALS PROVIDES A MORE BLOCKY STYLE.

To use this font in "font-family" declarations, use "Hachicro".

Reccomended fallback fonts are as follows:

  • OCR A Extended
  • OCR A
  • Goudy Stout
  • Impact
  • sans-serif

The following classes make use of this font as their default:

  • ut-damage
